# Development tools

> Docker image for building latex documents.

## Usage

Available from:
- [Dockerhub](https://hub.docker.com/r/voidbuffer/texlive)
- [Github](https://github.com/eliaskanelis/texlive)

```sh
docker pull voidbuffer/texlive
```

## Development

### Build

```sh
make
```

### Run

```sh
make run
```
